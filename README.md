# Aplicacion-Condominios

## Requisitos

El proyecto usa una `venv` local en `.venv/`.

## Instalacion

```bash
/usr/bin/python3 -m venv .venv
.venv/bin/python -m pip install -r requirements.txt
```

## Ejecucion

```bash
./run.sh
```

Si prefieres ejecutarlo manualmente:

```bash
.venv/bin/python main.py
```
